# Folder Assets

Tempatkan file-file berikut di folder ini:

## Audio Files
- `birthday-music.mp3` - Musik latar belakang instrumental
- `blow-sound.mp3` - Sound effect tiupan lilin

## Image Files (Opsional)
- `photo-enjelly.jpg` - Foto untuk ditampilkan
- `background-photo.jpg` - Foto untuk background

## Cara Menambahkan File

1. Download atau siapkan file audio dan foto Anda
2. Rename sesuai nama yang tercantum di atas
3. Copy file ke folder ini: `public/assets/`

## Catatan Penting

⚠️ **File audio tidak disertakan secara default!**

Anda perlu menambahkan file audio sendiri agar:
- Musik latar belakang dapat diputar
- Sound effect tiupan lilin berfungsi

Jika file tidak ada, aplikasi akan tetap berjalan normal tetapi tanpa audio.
